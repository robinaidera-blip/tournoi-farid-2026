Règlement

Points

  • Victoire : 4 points
  • Nul : 2 points
  • Défaite : 1 point
  • Forfait : score officiel 3-0, 0 point pour l'équipe forfait

Départage

  • Points → Différence de buts → Buts marqués → Confrontation directe → TAS

Format du tournoi

  • 16 équipes réparties en 2 poules de 8
  • Phase de poules : matchs aller simple, round-robin complet
  • Les 4 premiers de chaque poule se qualifient pour les demi-finales Champions
  • Les équipes 5-8 disputent le tournoi Europa
